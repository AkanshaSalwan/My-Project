Integrating Shopify

Step 01

Access Shopify Theme Editor:

  • Log in to your Shopify admin.
  • Navigate to Online Store Themes.

    • Step 02

    Edit Theme Code :

    • Click on the "Actions" dropdown for your active theme and select "Edit code."
    • Upload JS File:
    • Upload your loadChatbot.js file to your Shopify theme. You can place it in the "Assets" folder.

    Step 03

    Include JS in Theme:

    • In the "Sections" or "Templates" folder (depending on your theme structure), locate the theme file (e.g., theme.liquid or footer.liquid) where you want to include your JavaScript file.
    • Add the following code just before the closing </body> tag:
      <script src="{{ 'loadChatbot.js' | asset_url }}"></script>
      Replace loadChatbot.js with the actual name of your JavaScript file.

    Step 04

    Save Changes :

    Save the changes in the Shopify theme editor.

    Step 05

    Verify :

    • Visit your Shopify store and open the browser's developer tools (usually by right-clicking and selecting "Inspect").
    • Go to the "Console" tab and check for any errors. Ensure that your loadChatbot.js file is being loaded without errors.

    Step 06

    Test :

    Test the chat window functionality to ensure it appears as expected on your Shopify store.
